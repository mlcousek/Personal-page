# Personal Webpage

This is a modern design personal webpage project. Below are the details regarding the structure and usage of the project.

## Project Structure

```
personal-webpage
├── src
│   ├── assets
│   │   └── styles
│   │       └── main.css       # CSS styles for the webpage
│   ├── components
│   │   └── header.html        # Header component
│   │   └── footer.html        # Footer component
│   ├── index.html             # Main HTML file
│   └── scripts
│       └── main.js            # JavaScript functionality
├── package.json                # npm configuration file
└── README.md                   # Project documentation
```

## Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd personal-webpage
   ```
3. Install the dependencies:
   ```
   npm install
   ```

## Usage

- Open `src/index.html` in your web browser to view the personal webpage.
- Modify the CSS in `src/assets/styles/main.css` to change the appearance of the webpage.
- Update the header and footer components in `src/components/header.html` and `src/components/footer.html` as needed.
- Add any JavaScript functionality in `src/scripts/main.js`.
